## Visao geral

Este adaptador serve para ambientes edge sem wrapper pronto e entrega controle direto sobre o modulo de renderizacao.

## Exemplo

```ts
import { createOgHandler } from "@better-og/edge";
```

- Use `@better-og/edge` quando precisar fornecer o modulo Takumi manualmente.
