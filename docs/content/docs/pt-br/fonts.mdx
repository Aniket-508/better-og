## Visao geral

better-og facilita OG images multilíngues com tipografia consistente mesmo quando o canvas mistura varios sistemas de escrita.

- `resolveFontSetup` combina as fontes base com os fallbacks Noto corretos.
- A locale pode vir dos params ou de um callback de request.
