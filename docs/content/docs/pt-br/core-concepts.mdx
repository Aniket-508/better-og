## Visao geral

better-og usa sinais reais do request para identificar a plataforma e monta um plano de render consistente para o resto da pipeline.

- O request OG resolvido inclui plataforma, dimensoes, safe area e plano de layout.
- Voce pode forcar um preset com `?aspect_ratio=`.
