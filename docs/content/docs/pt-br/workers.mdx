## Visao geral

`@better-og/workers` entrega um ponto de partida rapido para imagens OG em runtimes do tipo worker sem muita configuracao manual.

## Exemplo

```ts
import { createOgHandler } from "@better-og/workers";
```

- A inicializacao do Takumi e do renderer e resolvida internamente no caso comum.
