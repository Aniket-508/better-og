## نظرة عامة

يتعامل better-og مع اكتشاف المنصة، وحساب الأبعاد، وتطبيع خطة العرض قبل أن يصل الطلب إلى مكون الصورة.

- يُطبع طلب OG المحلول نسبة العرض إلى الارتفاع والمنصة والمناطق الآمنة والتخطيط النهائي.
- يمكنك فرض الإعداد يدويًا عبر `?aspect_ratio=` إذا احتجت لذلك.
