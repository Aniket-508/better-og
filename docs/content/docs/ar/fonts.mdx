## نظرة عامة

يوفر better-og طبقة إعداد خطوط تضمن أن النصوص متعددة اللغات تظهر بشكل صحيح عبر الصور والمنصات المختلفة.

- يمكن لـ `resolveFontSetup` دمج الخطوط الأساسية مع خطوط Noto المناسبة للغة المطلوبة.
- يمكن استخراج اللغة من params أو من callback مخصص لكل طلب.
