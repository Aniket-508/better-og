## نظرة عامة

هذا المهايئ مناسب لبيئات Edge التي لا توفر غلافًا جاهزًا، لكنه يمنحك تحكمًا كاملاً في وحدة العرض.

## مثال

```ts
import { createOgHandler } from "@better-og/edge";
```

- استخدم `@better-og/edge` عندما تحتاج إلى تمرير وحدة Takumi بنفسك.
