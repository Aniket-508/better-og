## Ringkasan

Adaptor ini cocok untuk lingkungan edge tanpa wrapper siap pakai dan memberi Anda kontrol penuh atas modul renderer.

## Contoh

```ts
import { createOgHandler } from "@better-og/edge";
```

- Gunakan `@better-og/edge` saat Anda perlu menyuplai modul Takumi sendiri.
