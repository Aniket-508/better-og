## Ringkasan

better-og memudahkan gambar OG multibahasa tetap tampil rapi walau menggabungkan beberapa sistem tulisan dalam satu gambar.

- `resolveFontSetup` dapat menggabungkan font dasar dengan fallback Noto yang tepat.
- Locale bisa diambil dari params atau callback request.
