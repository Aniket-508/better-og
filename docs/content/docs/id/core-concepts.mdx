## Ringkasan

better-og membaca sinyal request untuk mengenali platform lalu membangun rencana render yang konsisten untuk pipeline berikutnya.

- Request OG yang sudah dinormalisasi berisi platform, dimensi, safe area, dan rencana layout.
- Anda dapat memaksa preset dengan `?aspect_ratio=` bila diperlukan.
