## Ringkasan

`@better-og/workers` memberi titik awal yang cepat untuk gambar OG pada runtime bergaya worker tanpa banyak konfigurasi manual.

## Contoh

```ts
import { createOgHandler } from "@better-og/workers";
```

- Inisialisasi Takumi dan setup renderer ditangani secara internal pada kasus umum.
