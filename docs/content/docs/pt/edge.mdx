## Visao geral

Este adaptador serve ambientes edge sem um wrapper pronto e da-lhe controlo direto sobre o modulo de renderizacao.

## Exemplo

```ts
import { createOgHandler } from "@better-og/edge";
```

- Use `@better-og/edge` quando precisar de fornecer o modulo Takumi manualmente.
