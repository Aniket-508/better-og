## Visao geral

better-og facilita imagens OG multilingues com tipografia estavel mesmo quando o mesmo canvas mistura varios sistemas de escrita.

- `resolveFontSetup` pode combinar as fontes base com os fallbacks Noto corretos.
- A locale pode vir dos params ou de um callback por request.
