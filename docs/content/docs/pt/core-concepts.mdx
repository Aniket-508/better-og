## Visao geral

better-og analisa sinais reais do request para descobrir a plataforma e cria um plano de render consistente para o resto da pipeline.

- O pedido OG resolvido inclui plataforma, dimensoes, safe area e plano de layout.
- Pode forcar um preset com `?aspect_ratio=`.
