## Vue d'ensemble

better-og facilite le rendu d'images multilingues coherentes, meme lorsque plusieurs systemes d'ecriture cohabitent.

- `resolveFontSetup` peut fusionner vos polices de base avec les variantes Noto appropriees.
- La locale peut etre resolue depuis les params ou un callback de requete.
