## Vue d'ensemble

better-og detecte la plateforme a partir des signaux de la requete puis construit un plan de rendu stable pour toute la pipeline.

- La requete OG resolue contient plateforme, dimensions, safe area et strategie de layout.
- Vous pouvez forcer un preset avec `?aspect_ratio=`.
