## Importer

```ts
import { createOgHandler } from "@better-og/node";
```

## Utilisation

L'adaptateur Node.js utilise le runtime natif Node.js de Takumi. Dans le cas général, vous passez simplement le composant et les options supplémentaires `ImageResponse`.

```ts
import { createOgHandler } from "@better-og/node";

const handler = createOgHandler({
  component: (
    <div
      style={{
        width: "100%",
        height: "100%",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
        background: "#1a1a2e",
        color: "#fff",
        fontSize: 48,
      }}
    >
      Node.js OG
    </div>
  ),
});

export default {
  fetch(request: Request) {
    return handler(request);
  },
};
```

## Quand l'utiliser

Utilisez `@better-og/node` quand :

- Vous exécutez dans un environnement Node.js (pas Next.js, pas Cloudflare Workers)
- Vous souhaitez utiliser le runtime natif Node.js de Takumi
