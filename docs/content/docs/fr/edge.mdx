## Vue d'ensemble

Cet adaptateur convient aux environnements edge sans wrapper pret a l'emploi et vous laisse le controle du module de rendu.

## Exemple

```ts
import { createOgHandler } from "@better-og/edge";
```

- Utilisez `@better-og/edge` lorsque vous devez fournir vous-meme le module Takumi.
