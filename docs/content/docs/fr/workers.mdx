## Vue d'ensemble

`@better-og/workers` fournit un point de depart rapide pour generer des images OG dans des runtimes de type worker.

## Exemple

```ts
import { createOgHandler } from "@better-og/workers";
```

- L'initialisation de Takumi et le renderer sont geres en interne dans le cas courant.
