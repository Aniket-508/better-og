## Огляд

better-og допомагає тримати стабільну типографіку в багатомовних OG-зображеннях, навіть коли в одному canvas змішані різні системи письма.

- `resolveFontSetup` може поєднати базові шрифти з потрібними Noto fallback.
- Locale можна взяти з params або request callback.
