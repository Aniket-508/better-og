## Огляд

Цей адаптер підходить для edge-середовищ без готової обгортки та дає повний контроль над модулем рендерингу.

## Приклад

```ts
import { createOgHandler } from "@better-og/edge";
```

- Використовуйте `@better-og/edge`, якщо модуль Takumi треба передати вручну.
