## Огляд

`@better-og/workers` дає швидкий старт для OG images у worker-подібних runtime без зайвої ручної конфігурації.

## Приклад

```ts
import { createOgHandler } from "@better-og/workers";
```

- Ініціалізація Takumi та налаштування renderer зазвичай виконуються всередині адаптера.
