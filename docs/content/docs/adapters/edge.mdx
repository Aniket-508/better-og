title: Generic Edge
description: Low-level Takumi WASM adapter for generic edge runtimes.
---

## Import

```ts
import { createEdgeOgHandler } from "better-og/edge";
```

## Usage

This adapter requires an explicit Takumi WASM module. It does not use route params for locale resolution -- provide a `localeFromRequest` callback instead.

```ts
import { createEdgeOgHandler } from "better-og/edge";

const handler = createEdgeOgHandler({
  module: () => import("@takumi-rs/wasm/next"),
  component: (ogContext) => (
    <div style={{
      width: "100%",
      height: "100%",
      display: "flex",
      alignItems: "center",
      justifyContent: "center",
    }}>
      Edge OG
    </div>
  ),
  localeFromRequest: (req) => {
    const url = new URL(req.url);
    return url.searchParams.get("locale") ?? undefined;
  },
});
```
