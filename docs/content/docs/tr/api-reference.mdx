## Genel bakis

Bu sayfa platform-aware OG route'lari icin en sik kullanilan preset, function ve type'lari toplar.

## Core (`@better-og/core`)

```ts
const STANDARD: AspectRatioPreset;
const SQUARE: AspectRatioPreset;
const PORTRAIT: AspectRatioPreset;
const INSTAGRAM: AspectRatioPreset;

declare function detectPlatform(request: Request): Promise<DetectedPlatform>;
declare function resolveOgRequest(request: Request): Promise<ResolvedOgRequest>;
declare function resolveFontSetup(
  options: ResolveFontSetupOptions
): Promise<ResolvedFontSetup>;
```
