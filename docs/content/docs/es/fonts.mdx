## Resumen

better-og facilita que una misma imagen muestre bien varios sistemas de escritura sin romper la tipografia.

- `resolveFontSetup` puede combinar tus fuentes base con las variantes Noto adecuadas.
- El locale puede salir de params o de un callback por solicitud.
