---

## Resumen

Este adaptador encaja en entornos Edge sin un wrapper listo y te deja controlar directamente el modulo de render.

## Ejemplo

```ts
import { createOgHandler } from "@better-og/edge";
```

- Usa `@better-og/edge` cuando necesites proporcionar tu propio modulo de Takumi.
