## Resumen

`@better-og/next` empaqueta fuentes, proxy y presets para crear rutas OG conscientes de la plataforma directamente en Next.js.

## Ejemplo

```tsx title="app/og/[lang]/route.tsx"
import { resolveFontSetup } from "@better-og/core";
import {
  createOgRouteHandler,
  loadGoogleFontForImageResponse,
} from "@better-og/next";

export const runtime = "nodejs";

const fontSetup = await resolveFontSetup({
  baseFonts: await loadGoogleFontForImageResponse({
    family: "Geist",
    weights: [400, 700],
  }),
});

export const GET = createOgRouteHandler({
  component: (
    <div
      style={{
        width: "100%",
        height: "100%",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
        fontFamily: fontSetup.families.base,
      }}
    >
      Platform-aware OG
    </div>
  ),
  baseFonts: fontSetup.fonts,
});
```

## Notas

- El renderer por defecto es `next/og`.
- Puedes cambiar a `renderer: "takumi"` cuando lo necesites.
