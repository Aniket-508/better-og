## Resumen

`@better-og/workers` te da un punto de partida rapido para generar imagenes OG en runtimes tipo worker sin demasiada configuracion manual.

## Ejemplo

```ts
import { createOgHandler } from "@better-og/workers";
```

- La inicializacion de Takumi y el renderer se resuelven internamente en el caso comun.
