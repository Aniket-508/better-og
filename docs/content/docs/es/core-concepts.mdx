## Resumen

better-og detecta la plataforma a partir de senales reales de la peticion y prepara un plan de render consistente para el resto del flujo.

- La solicitud OG resuelta incluye plataforma, dimensiones, safe area y estrategia de layout.
- Puedes forzar un preset manualmente con `?aspect_ratio=`.
