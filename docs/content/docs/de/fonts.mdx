## Uberblick

better-og hilft dir dabei, typografisch stabile OG-Bilder zu rendern, auch wenn mehrere Schriftsysteme in einem Bild vorkommen.

- `resolveFontSetup` kann deine Basisfonts mit passenden Noto-Fallbacks zusammenfuhren.
- Das Locale kann aus Params oder einer Request-Callback kommen.
