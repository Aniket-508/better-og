## Uberblick

Dieser Adapter passt zu Edge-Umgebungen ohne fertige Integration und gibt dir volle Kontrolle uber das Renderer-Modul.

## Beispiel

```ts
import { createOgHandler } from "@better-og/edge";
```

- Nutze `@better-og/edge`, wenn du das Takumi-Modul selbst bereitstellst.
