---

## Importieren

```ts
import { createOgHandler } from "@better-og/node";
```

## Verwendung

Der Node.js-Adapter verwendet Takumis native Node.js-Laufzeit. Im Allgemeinen übergibst du nur die Komponente und beliebige zusätzliche `ImageResponse`-Optionen.

```ts
import { createOgHandler } from "@better-og/node";

const handler = createOgHandler({
  component: (
    <div
      style={{
        width: "100%",
        height: "100%",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
        background: "#1a1a2e",
        color: "#fff",
        fontSize: 48,
      }}
    >
      Node.js OG
    </div>
  ),
});

export default {
  fetch(request: Request) {
    return handler(request);
  },
};
```

## Wann man es verwendet

Verwende `@better-og/node` wenn:

- Du in einer Node.js-Umgebung läufst (nicht Next.js, nicht Cloudflare Workers)
- Du Takumis native Node.js-Laufzeit verwenden möchtest
