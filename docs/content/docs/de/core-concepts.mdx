## Uberblick

better-og erkennt anhand echter Request-Signale die aufrufende Plattform und erstellt daraus einen konsistenten Render-Plan.

- Das normalisierte OG-Request-Objekt enthalt Plattform, Dimensionen, Safe Area und Layout-Plan.
- `?aspect_ratio=` kann ein Preset manuell uberschreiben.
