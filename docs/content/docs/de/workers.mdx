## Uberblick

`@better-og/workers` liefert einen schnellen Einstieg fur OG-Bilder in worker-ahlichen Umgebungen ohne viel Boilerplate.

## Beispiel

```ts
import { createOgHandler } from "@better-og/workers";
```

- Takumi-Initialisierung und Renderer-Einrichtung werden im Standardfall intern erledigt.
