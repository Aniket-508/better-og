## Обзор

better-og помогает сохранять стабильную типографику в многоязычных OG-изображениях даже при смешении разных систем письма.

- `resolveFontSetup` умеет объединять базовые шрифты с подходящими Noto fallback.
- Locale можно получить из params или request callback.
