## Обзор

`@better-og/workers` дает быстрый старт для OG images в worker-подобных runtime без лишней ручной настройки.

## Пример

```ts
import { createOgHandler } from "@better-og/workers";
```

- Инициализация Takumi и настройка renderer обычно выполняются внутри адаптера.
