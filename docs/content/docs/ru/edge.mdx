## Обзор

Этот адаптер подходит для edge-сред без готовой обертки и дает полный контроль над модулем рендера.

## Пример

```ts
import { createOgHandler } from "@better-og/edge";
```

- Используйте `@better-og/edge`, если модуль Takumi нужно передать вручную.
