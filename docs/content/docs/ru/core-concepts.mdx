## Обзор

better-og анализирует сигналы request, определяет платформу и строит единый план рендера для всей дальнейшей цепочки.

- Нормализованный OG request включает платформу, размеры, safe area и план layout.
- Preset можно принудительно задать через `?aspect_ratio=`.
