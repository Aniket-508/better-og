## Panoramica

better-og legge i segnali della richiesta per capire quale piattaforma sta recuperando l'immagine e prepara un piano di render coerente.

- La richiesta OG risolta include piattaforma, dimensioni, safe area e piano di layout.
- Puoi forzare un preset con `?aspect_ratio=`.
