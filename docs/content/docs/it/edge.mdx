## Panoramica

Questo adattatore e adatto agli ambienti edge senza wrapper pronto e ti lascia il controllo completo del modulo di render.

## Esempio

```ts
import { createOgHandler } from "@better-og/edge";
```

- Usa `@better-og/edge` quando devi fornire tu stesso il modulo Takumi.
