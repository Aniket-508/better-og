## Panoramica

better-og rende piu semplice produrre immagini multilingue stabili anche quando nello stesso canvas convivono piu sistemi di scrittura.

- `resolveFontSetup` puo unire i font di base con i fallback Noto corretti.
- La locale puo arrivare da params o da un callback di richiesta.
