## Panoramica

`@better-og/workers` fornisce un punto di partenza rapido per immagini OG in runtime stile worker senza molta configurazione manuale.

## Esempio

```ts
import { createOgHandler } from "@better-og/workers";
```

- L'inizializzazione di Takumi e del renderer e gestita internamente nel caso comune.
