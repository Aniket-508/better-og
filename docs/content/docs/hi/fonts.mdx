## ओवरव्यू

better-og बहुभाषी OG images को stable typography के साथ render करना आसान बनाता है, भले एक image में कई scripts हों।

- `resolveFontSetup` base fonts को सही Noto fallback के साथ जोड़ सकता है।
- locale params या request callback से निकाला जा सकता है।
