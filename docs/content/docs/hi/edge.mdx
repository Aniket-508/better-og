## ओवरव्यू

यह adapter उन edge environments के लिए है जहां ready-made wrapper नहीं है और आपको renderer module पर सीधा control चाहिए।

## उदाहरण

```ts
import { createOgHandler } from "@better-og/edge";
```

- जब आपको Takumi module खुद देना हो, तब `@better-og/edge` इस्तेमाल करें।
