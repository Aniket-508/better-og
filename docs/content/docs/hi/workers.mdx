## ओवरव्यू

`@better-og/workers` worker-style runtimes में OG images जल्दी शुरू करने के लिए सरल entry point देता है।

## उदाहरण

```ts
import { createOgHandler } from "@better-og/workers";
```

- common case में Takumi init और renderer setup अंदर ही हो जाता है।
