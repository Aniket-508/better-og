## Overblik

Denne adapter passer til edge-miljoer uden en faerdig wrapper og giver dig fuld kontrol over renderer-modulet.

## Eksempel

```ts
import { createOgHandler } from "@better-og/edge";
```

- Brug `@better-og/edge`, naar du selv skal levere Takumi-modulet.
