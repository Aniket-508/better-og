## Overblik

better-og afgor hvilken platform der henter billedet og bygger en ensartet renderplan for resten af pipelinen.

- Det normaliserede OG-request indeholder platform, dimensioner, safe area og layoutplan.
- Du kan tilsidesaette preset manuelt med `?aspect_ratio=`.
