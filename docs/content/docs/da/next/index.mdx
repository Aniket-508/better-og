## Overblik

`@better-og/next` giver en direkte vej til platform-bevidste OG-ruter i Next.js med fonte, proxy og presets samlet et sted.

## Eksempel

```tsx title="app/og/[lang]/route.tsx"
import { resolveFontSetup } from "@better-og/core";
import {
  createOgRouteHandler,
  loadGoogleFontForImageResponse,
} from "@better-og/next";

export const runtime = "nodejs";

const fontSetup = await resolveFontSetup({
  baseFonts: await loadGoogleFontForImageResponse({
    family: "Geist",
    weights: [400, 700],
  }),
});

export const GET = createOgRouteHandler({
  component: (
    <div
      style={{
        width: "100%",
        height: "100%",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
        fontFamily: fontSetup.families.base,
      }}
    >
      Platform-aware OG
    </div>
  ),
  baseFonts: fontSetup.fonts,
});
```

## Noter

- Standardrendereren er `next/og`.
- Skift til `renderer: "takumi"`, hvis du har brug for det.
