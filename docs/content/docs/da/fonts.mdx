## Overblik

better-og gor det nemt at holde typografi stabil, selv nar det samme billede skal vise flere skriftsystemer.

- `resolveFontSetup` kan kombinere dine basisfonte med korrekte Noto-fallbacks.
- Locale kan findes fra params eller en request-callback.
