## Overblik

`@better-og/workers` giver et hurtigt startpunkt til OG-billeder i worker-lignende miljoer uden meget manuel opsaetning.

## Eksempel

```ts
import { createOgHandler } from "@better-og/workers";
```

- Takumi-initialisering og renderer-haandtering klares internt i standardtilfaeldet.
